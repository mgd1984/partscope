# PartScope

Standalone Three.js CAD review app for mechanical assemblies.

## Setup

```bash
npm install
npm run dev
```

The app opens with the bundled demo assembly and can also load local `.stl` files directly in the browser.

## Build

```bash
npm run build
```

## Included models
